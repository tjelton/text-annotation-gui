# Annotation configuration — Named Entity Recognition (NER) example
#
# Format:
#   Label:  <name>  <key>  <colour>
#
# <name>   : the classification label (appears in .annotations files as "label:<name>")
# <key>    : single keyboard character to apply this label (must not be [ ] u)
# <colour> : colour name or hex (#RRGGBB)
#
# Supported colour names:
#   red, blue, green, yellow, magenta, purple, cyan, orange, pink, brown,
#   white, black, lime, indigo, teal, coral, navy
#
# Press the key while text is selected to apply/toggle the label.
# Press 'u' to remove all labels from the current selection.
# Press ']' for next file, '[' for previous file.
# Ctrl+S to save, Ctrl+Z to undo, Ctrl+Q to quit.
